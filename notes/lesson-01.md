## Questions

### How does context window size affect memory?
Larger context windows allow the model to process and remember more information within a conversation or document. However, LLMs do not truly “remember” like humans — they only use the currently available context.

### How can hallucinations be reduced?
Hallucinations can be reduced with better prompts, grounding techniques like RAG, validation workflows, tool usage and human review.

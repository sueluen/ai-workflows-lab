# Lesson 01 — Introduction to Generative AI

## Core Concepts
- Generative AI creates new content
- LLMs predict next tokens
- Transformers use attention mechanisms
- Models are probabilistic, not truly intelligent

## Important Takeaways
- Prompt = interface
- AI is strongest inside workflows
- Hallucinations are a major limitation

## Real-World Applications
- AI-powered CRM assistant
- AI-powered quote generation assistant
- Legal document summarization
- WhatsApp intake assistant

## Ideas / Thoughts
- Build AI-powered legal intake workflow
- Experiment with AI proposal generation
- Investigate RAG for PDF search

## Questions

### How does context window size affect memory?
Larger context windows allow the model to process and remember more information within a conversation or document. However, LLMs do not truly “remember” like humans — they only use the currently available context.

### How can hallucinations be reduced?
Hallucinations can be reduced with better prompts, grounding techniques like RAG, validation workflows, tool usage and human review.
